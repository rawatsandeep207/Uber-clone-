🚀 Live Project Overview

The Uber Clone Web Application is a full-stack cab booking platform developed using HTML, CSS, PHP, and MySQL.

It simulates real-world ride booking functionality with separate User and Admin panels, authentication system, and complete CRUD operations.

This project demonstrates backend logic, database handling, and role-based access control.
